CSS Syntax

A CSS rule consists of a selector and declaration block :

h1   {color:red; font-size:13px;}

h1 : Selector

color : Property

red : Value

Types of selectors

  1. Element Selector : Extension used to save external CSS files is .css

  2. ID Selector : CSS : It describes how the HTML elements are to be displayed on screen or in other media.

  3. Class Selector : CSS saves lots of work. Multiple pages can be styled using CSS.

  4. Universal Selector : Font Family Provided.

  5. Grouping Selector : You can apply same to all the type of selectors just by comma seprated selectors.

Types of stylesheets

  1. Inline CSS : Most prioritized type of stylesheet written within HTML tag. [Not recommended creates messy code]
  2. Internal CSS : Least prioritized type of stylesheet written within HTML file. [Usefull for single page designing not recommended for project].
  3. External CSS : Second prioritized type of stylesheet written outside HTML file using .css extension file has been created and linked to HTML page using <link>.

CSS comments can be written as /* */ for single and multi lines both are same.

CSS Errors may arise when you provide wrong propetry name or wrong values to correct properties.

Colons, Braces errors may also encountered.

Tips : Use code editor, check styling frequently

Types of colors

  1. Direct Color Names Like : Mediumseagreen, Violet, Blue
  2. Using rgb like : RED : 200, GREEN : 0, BLUE : 0
  3. Using rgba like : RED : 0, GREEN : 0, BLUE : 250, ALPHA (OPACITY) : 0.5
  4. Using hex values like : #ff0000 - ff:MAX 00:MIN
  5. Using hexa values like : #ff - RED MAX, 00 - BLUE GREEN 0, Opacity 80%
  6. Using hsl values like : HUE : 180 o, SATURATION : 50%, LIGHTNESS : 20%
  7. Using hsla values like : HUE : 180 o, SATURATION : 50%, LIGHTNESS : 20%, ALPHA (OPACITY) : 0.8

Box Model

Calculation for CSS Box Model

Given width = 200px, height = 200px, border = 10px, margin = 60px, padding = 50px

  • Total Width = Width (200px) + Left Padding (50px) + Right Padding (50px) + Left Border (10px) + Right Border (10px) = 320px
  • Total Height = Height (200px) + Top Padding (50px) + Bottom Padding (50px) + Top Border (10px) + Bottom Border (10px) = 320px
  • Total Size = 320x320
