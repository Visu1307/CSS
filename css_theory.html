CSS Docs

CSS

It is used to style web pages.

Structure of webpages are defined by HTML.

Styles of webpages are defined by CSS.

CSS3 (NOT a single version)

This is where the big change happened.

CSS3 has been divided into seperate modules.

Each module evolves seperately.

Major CSS 3 Modules

  • Media Queries (Responsive Design)
  • Flexbox
  • Grid
  • Animations & Transitions
  • Border-radius
  • Box-shadow
  • Gradients
  • Transform (2D / 3D)

Mordern Features

  • CSS Grid (Layout system)
  • Flexbox (One-dimensional layout)
  • Custom Properties (CSS Variables)
  • :is(), :where()
  • clamp(), min(), max()
  • Container Queries
  • Logical properties (margin-inline, etc.)

Difference between Old CSS & Current CSS

Feature Old CSS Mordern CSS
Layout Float, Table Flexbox, Grid
Responsiveness No Media + Container Queries
Reuseability No CSS Variables
Animations No Native animations
Browser support Inconsistent Standardized
Maintainability Hard Easy Scalable
