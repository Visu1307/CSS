CSS Docs

CSS

It is used to style web pages.

Structure of webpages are defined by HTML.

Styles of webpages are defined by CSS.

CSS3 (NOT a single version)

This is where the big change happened.

CSS3 has been divided into seperate modules.

Each module evolves seperately.

Major CSS 3 Modules

  • Media Queries (Responsive Design)
  • Flexbox
  • Grid
  • Animations & Transitions
  • Border-radius
  • Box-shadow
  • Gradients
  • Transform (2D / 3D)

Mordern Features

  • CSS Grid (Layout system)
  • Flexbox (One-dimensional layout)
  • Custom Properties (CSS Variables)
  • :is(), :where()
  • clamp(), min(), max()
  • Container Queries
  • Logical properties (margin-inline, etc.)

Difference between Old CSS & Current CSS

Feature Old CSS Mordern CSS
Layout Float, Table Flexbox, Grid
Responsiveness No Media + Container Queries
Reuseability No CSS Variables
Animations No Native animations
Browser support Inconsistent Standardized
Maintainability Hard Easy Scalable

Difference between CSS Display Flex and Display Grid

Property CSS Grid CSS Flexbox
Dimensions Two-dimensional (rows and columns) One-dimensional (either a row or a column)
Approach Layout-first; you define the grid structure and place items into it Content-first; items flow naturally and flex to fit the available space
Use Case Complex page layouts, dashboards, or designs requiring precise 2D control Small components like navigation bars, aligning items within a container, or simple lists
Control Parent container has complete control over child placement and sizing Child items have more control over their own growth and shrinking behavior
Gaps gap property provides spacing between both rows and columns gap property can be used, but the core model is a single line, so behavior can be less intuitive with wrapping
