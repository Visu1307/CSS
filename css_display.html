CSS Display

  • display : block - It takes content height and full width.

    • If content's height keeps increasing then height of block parent will also be increasing.

    Examples of block level elements : 
                    <div>
                <h1> - <h6>
                <p>
                <form>
                <header>
                <footer>
                <section>
  • display : inline - It occupies only required height and width.
    • You can easily change the default behaviour of any element like this is an div but display is inline it occupies only required height and width not full.

    Height and width properties will won't work in inline display style.

    Examples of inline level elements : 

                    <a>
                <span>
                <img>
  • display : inline-block : It allows you to use inline abilites while allowing height and width properties.
    • Hello World, How Are You? I am Fine!
