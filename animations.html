Animations

CSS allows animation of HTML elements without using JavaScript!

An animation lets an element gradually change from one style to another.

@keyframes identifier{ from {proeprty;} to {property;} }

Animated Color Change

animation-name : identifier - It identifies which animation should be applied to particular element.

animation-duration : seconds - It ensures that animation should run for provided duration.

animation-iterate-count : n/infinite - It defines number of iterations for animation.

Hover over the element to see the change.

CSS animation-fill-mode property

CSS animations do not affect an element before the first keyframe is played or after the last keyframe is played. The animation-fill-mode property can override this behavior.

animation-fill-mode : (none/forwards/backwards/both) - It specifies a style for the target element when the animation is not playing (before it starts, after it ends, or both).

It preserves normal style.

animation-fill-mode : forwards - It preserves the last keyframe style applied that is BLACK color in our case.

It preserves LAST keyframe style applied.

animation-fill-mode : backwards - It preserves the first keyframe style applied that is RED color in our case.

NOTICEABLE only when animation-delay is given.

It preserves FIRST keyframe style applied.

animation-fill-mode : both - It preserves the both first and last keyframe style applied that is RED AND BLACK color in our case.

It preserves FIRST & LAST keyframes style applied.
