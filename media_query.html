Media Queries

CSS media queries allow you to apply styles based on the characteristics of a device or the environment displaying the web page.

CSS media queries are essential for creating responsive web pages.

The CSS @media rule is used to add media queries to your style sheet.

Media Query Syntax

@media [not] media-type and (media-feature: value) and (media-feature: value) {
/* CSS rules to apply */
}

CSS Media Types

  • all - Default value if not specified works in all devices.
  • print - Here you can specify how to display and style elements in print preview mode.
  • screen - Specify deisgn rules for computer, tablet, mobile etc. devices.

media-type : all

@media all { .testAll{ background-color: yellow; } }
Hello World

media-type : all

@media not all { .testAll{ background-color: red; } }
Hello World

media-type : print

@media print { .testPrint{ background-color: blue; color: white; font-size: large; } }
Hello World

media-type : screen

It is used along with certain parameters such as :

  • max-height : Maximum height of the viewport
  • min-height : Minimum height of the viewport
  • max-width : Maximum width of the viewport
  • min-width : Minimum width of the viewport

max-height - 600px, Style <= 600px < Default.

@media screen and (max-height : 600px){ .maxHeight{ background-color: brown; color: white; } }
Max Height 600px

min-height - 600px, Default < 600px <= Style.

@media screen and (min-height : 600px){ .minHeight{ background-color: violet; color: black; } }
Min Height 600px

min-height - 600px AND max-height - 800px, Default < 600px <= Style <= 800 < Default.

@media screen and (min-height : 600px) and (max-height : 800px) { .minmaxHeight{ background-color: black; color: white; } }
Min Max Height

max-width - 600px, Style <= 600px < Default

@media screen and (max-width : 600px){ .maxWidth{ background-color: aqua; color: black; } }
Max Width 600px

min-width - 600px, Default < 600px <= Style

@media screen and (min-width : 600px){ .minWidth{ background-color: chocolate; color: black; } }
Min Width 600px

min-width - 600px AND max-width - 800px, Default < 600px <= Style <= 800px < Default

@media screen and (min-width : 600px){ .minWidth{ background-color: purple; color: white; } }
Min 600px Width
Max 800px Width

Media Queries for Screen Orientation

orientation : landscape

@media screen and (orientation : landscape) { .land{ background-color: powderblue; } }
Hello World!
It's Landscape

orientation : portrait

@media screen and (orientation : portrait) { .port{ background-color: orange; } }
Hello World!
It's Portrait
